# Guide Utilisateur

## Création d'un Projet
1. Ouvrez ExeOutput Studio.
2. Cliquez sur "Nouveau Projet".
3. Sélectionnez votre dossier source contenant vos fichiers `.php`.
4. Configurez les options de compilation (version PHP, port, etc).
5. Cliquez sur "Compiler".

## Distribution
Le fichier EXE généré se trouve dans le dossier `dist` de votre projet.
Vous pouvez le distribuer tel quel, il inclut tout le nécessaire pour s'exécuter.
